Fill in the details below to post a new job opportunity

Job Information

Job title is required Job title must be at least 3 characters
Position is required Position must be at least 2 characters
Qualification is required Qualification must be at least 5 characters
Organization is required Organization must be at least 2 characters
Organization is required
Description is required Description must be at least 10 characters
Application link is required

API Integration for Developers

Integrate job posting functionality into your application using our API endpoints.

POST - Add Job

POST /api/jobs/add

Request Body:

{{ '{' }}
  "title": "Software Developer",
  "position": "Senior",
  "qualification": "Bachelor's in CS, 3+ years experience",
  "organization": "Tech Corp",
  "location": "Bangalore",
  "description": "We are hiring senior developers...",
  "applyUrl": "https://example.com/apply"
{{ '}' }}

Response (Success):

{{ '{' }}
  "success": true,
  "message": "Job posted successfully",
  "data": {{ '{' }}
    "id": 123,
    "title": "Software Developer",
    "position": "Senior",
    "qualification": "Bachelor's in CS, 3+ years experience",
    "organization": "Tech Corp",
    "location": "Bangalore",
    "description": "We are hiring senior developers...",
    "applyUrl": "https://example.com/apply",
    "createdAt": "2026-06-09T10:30:00Z"
  {{ '}' }}
{{ '}' }}

Error Response:

{{ '{' }}
  "success": false,
  "message": "Validation error",
  "errors": {{ '{' }}
    "title": "Title is required",
    "qualification": "Qualification must be at least 5 characters"
  {{ '}' }}
{{ '}' }}
